* CTickView - 打勾
* ToggleButton - Button按钮(滑动可选)
* ReplyBgView - 回复消息背景
* VSView - 对比
* HoleBgView - 镂空背景
* ArcMenu - 卫星菜单
* LrcView - 歌词控件
* SideBar - Sort联系人排序、
* TCHeartLayout - 点赞动效
